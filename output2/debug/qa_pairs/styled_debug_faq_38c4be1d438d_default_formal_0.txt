Original Question: Preciso fazer o Enade para me formar em Engenharia de Computação?
Answer: A obrigatoriedade do Enade para Engenharia de Computação depende do ano.

*   O exame não é aplicado todo ano para este curso.
*   Se houver Enade para Engenharia de Computação no ano em que você for se formar, **sim, você é obrigado(a) a fazer**.
*   Se não houver Enade para o curso no seu ano de formatura, você será **automaticamente dispensado(a)**, e a pendência será removida do seu histórico.

Especificamente para o ano de **2024**, não haverá aplicação do Enade para os formandos de Engenharia de Computação da UnB.
