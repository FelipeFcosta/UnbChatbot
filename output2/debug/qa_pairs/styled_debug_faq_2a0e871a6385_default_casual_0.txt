Original Question: Existem pré-requisitos para poder iniciar o Projeto Final em Engenharia de Computação?
Answer: Sim, para iniciar o Projeto Final em Engenharia de Computação é preciso atender a alguns pré-requisitos, como ter completado uma porcentagem mínima da carga horária total do curso. Recomenda-se consultar o Regulamento de TCC do curso para verificar todas as exigências.
