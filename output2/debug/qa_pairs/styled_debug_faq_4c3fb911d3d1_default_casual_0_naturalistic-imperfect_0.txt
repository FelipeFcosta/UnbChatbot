Original Question: Tem alguma sugestão de ordem pra fazer as disciplinas de Engenharia de Computação a cada semestre?
Answer: Sim, existem sugestões de fluxo curricular que podem lhe orientar sobre uma ordem recomendada para cursar as disciplinas de Engenharia de Computação a cada semestre. Você pode consultá-las aqui:

*   [Sugestão para o currículo 1741/1](https://www.ene.unb.br/joaoluiz/engcomputacao_fluxo_sugerido_1741-1.pdf)
*   [Sugestão para o currículo 1741/2](https://www.ene.unb.br/joaoluiz/engcomputacao_fluxo_sugerido_1741-2.pdf)

Lembre-se que o número do seu currículo consta no seu histórico escolar.

É importante notar que você tem liberdade para escolher as disciplinas que cursará a cada semestre, desde que os pré-requisitos de cada uma sejam respeitados. Além disso, os fluxos sugeridos podem listar muitas disciplinas optativas; talvez não seja necessário cursar todas elas, pois a carga horária optativa pendente também é reduzida ao cursar eletivas (módulo livre) ou através de equivalências.
